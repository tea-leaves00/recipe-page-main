### What I learned

I learned more about how to use 'max-width' vs width, and why using 'height' is not always a great option, and to rather let content and padding define height of a box. I also have improved drastically at my implementation of flexbox. My semantic HTML is really coming together as well, used the article and section tags to the best of my ability. This was also my first time implementing the <HR> tag for visual line breaks

### Continued development

I'll continue to learn more about which CSS properties are best, and when to let content define the size of a box.
